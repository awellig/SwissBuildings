System Overview

The SwissBuild Nexus implements a comprehensive suite of algorithms for analyzing building characteristics, solar potential, environmental conditions, and energy consumption across Switzerland. The system leverages multiple authoritative data sources and implements industry-standard calculations to provide accurate, actionable insights.

Architecture Overview

The backend services are organized into specialized modules, each handling specific aspects of building analysis. All recommendation logic and economic calculations are performed in the backend to ensure consistency, accuracy, and prevent client-side manipulation.

  • Enhanced Solar Estimation Service: NASA POWER + PVGIS-based solar potential calculations
  • Building Data Service: Swiss Federal Building Registry integration
  • Environment Service: Air quality and weather data from NABEL/MeteoSwiss
  • Energy Service: Energy consumption analysis using NILM methodologies
  • Solar Service: Integration layer combining multiple solar data sources

Data Flow Architecture

Backend Processing (Authoritative)

  • NASA POWER satellite data retrieval and processing
  • Swiss Building Registry data extraction and validation
  • Economic viability calculations with Swiss market rates
  • Payback period and ROI analysis
  • Recommendation classification (excellent/good/moderate/poor)

Frontend Display (Presentation Only)

  • Data visualization and user interface
  • No economic calculations or recommendations
  • Direct display of backend-calculated values
  • User interaction and map integration

Solar Estimation Algorithms

Enhanced Swiss Solar Estimation

Service: EnhancedSwissSolarEstimationService

Primary Method: Multi-source satellite and ground-based data fusion

Algorithm Overview

The enhanced solar estimation algorithm uses NASA POWER satellite data as the primary irradiation source, combined with Swiss building characteristics and PVGIS methodologies to provide highly accurate solar potential calculations.

Calculation Steps

  1. NASA POWER Data Retrieval: 4-year satellite averages (2020-2023) for precise coordinates
  2. PVGIS Validation: EU Photovoltaic Geographical Information System cross-validation
  3. Swiss Building Analysis: Federal Building Registry roof characteristics
  4. Optimized Installation Design: Panel placement and efficiency calculations
  5. Economic Assessment: Swiss market-specific financial modeling with 2024 rates

Roof Area Calculation

Ground Floor Area = Total Floor Area ÷ Number of Floors
Roof Area = Ground Floor Area × Roof Multiplier
Suitable Area = Roof Area × Suitability Factor
Building Type Roof Multiplier Rationale
Residential (1010, 1020) 0.90 Simple roof structures, minimal complications
Industrial/Warehouse (1262, 1263) 0.95 Large, flat roofs ideal for solar installations
Educational (1311, 1312) 0.80 Complex roof structures, multiple sections
Default 0.85 Standard Swiss building average

Suitability Factor Calculation

Roof Area Suitability Factor Justification
≥ 200m² 0.70 Large roofs, minimal edge effects
100-199m² 0.65 Medium roofs, good utilization
< 50m² 0.55 Small roofs, significant edge effects

Panel Efficiency

Modern Panel Efficiency: 220W/m² (2024 Swiss market standard)

Potential Power (kWp) = Suitable Area (m²) × 0.22 (kW/m²)

NASA POWER Solar Irradiation Model

Primary solar irradiation data source using NASA POWER satellite measurements for precise location-specific calculations.

Data Source Characteristics

Parameter Value Details Validation
Spatial Resolution 0.5° × 0.625° Global grid, ~50km resolution Validated against ground stations
Temporal Coverage 2020-2023 4-year average for reliability Accounts for yearly variations
Data Parameter ALLSKY_SFC_SW_DWN All Sky Surface Shortwave Downward Irradiance Includes cloud effects and atmospheric conditions
Swiss Irradiation Range 900-1400 kWh/m²/year Location-dependent satellite measurements Consistent with MeteoSwiss ground data

Algorithm Implementation

Monthly Average = Σ(Daily Irradiance × Days in Month) for 2020-2023
Annual Irradiation = Σ(Monthly Averages × Days per Month)
Advantages over Regional Estimates: NASA POWER provides actual measured satellite data for each specific location rather than interpolated regional values, ensuring accuracy for precise coordinates.

Financial Calculation

Annual Savings = (Self-Consumed Energy × Electricity Price) + (Fed-in Energy × Feed-in Tariff)
Effective Cost = Installation Cost - (Installation Cost × Tax Benefit)
Payback Period = Effective Cost ÷ Annual Savings

Authoritative Data Sources

NASA POWER (Primary Solar Data Source)

Purpose: Precise location-based solar irradiance measurements

API Endpoint: https://power.larc.nasa.gov/api/temporal/monthly/point

Parameters: ALLSKY_SFC_SW_DWN (All Sky Surface Shortwave Downward Irradiance)

Resolution: 0.5° × 0.625° global grid, monthly averages

Time Period: 2020-2023 (4-year average for reliability)

Accuracy: Validated against ground-based pyranometer measurements

Swiss Coverage: Complete coverage with 900-1400 kWh/m²/year measured values

Documentation: NASA POWER Documentation

Swiss Federal Building Register (GWR)

Purpose: Official building data for Switzerland

API Endpoint: https://api3.geo.admin.ch/rest/services/api/MapServer

Layer: ch.bfs.gebaeude_wohnungs_register

Data Coverage: All buildings in Switzerland with EGID

Update Frequency: Continuous updates from cantonal authorities

Documentation: BFS Building Register

PVGIS (Photovoltaic Geographical Information System)

Purpose: European solar potential calculations

API Endpoint: https://re.jrc.ec.europa.eu/api/PVcalc

Database: SARAH-2 satellite data (EU Copernicus)

Resolution: 0.05° × 0.05° European grid

Methodology: Validated against ground measurements

Documentation: PVGIS Documentation

Swiss Solar Atlas (Sonnendach)

Purpose: Official Swiss roof-level solar potential

API Endpoint: https://api3.geo.admin.ch

Layer: ch.bfe.solarenergie-eignung-daecher

Resolution: Building-level precision for entire Switzerland

Authority: Swiss Federal Office of Energy (SFOE)

Documentation: Sonnendach Portal

NABEL Air Quality Network

Purpose: Swiss air pollution monitoring

API Endpoint: https://www.bafu.admin.ch/bafu/en/home/topics/air/state/data.html

Network: 16 automated stations across Switzerland

Parameters: NO₂, PM₁₀, PM₂.₅, O₃, SO₂

Update Frequency: Hourly measurements

Documentation: NABEL Documentation

MeteoSwiss Weather Network

Purpose: Official Swiss meteorological data

API Endpoint: https://data.geo.admin.ch/ch.meteoschweiz.messwerte-aktuell/

Network: 200+ automated weather stations

Parameters: Temperature, humidity, pressure, wind, precipitation

Update Frequency: 10-minute intervals

Documentation: MeteoSwiss Documentation
